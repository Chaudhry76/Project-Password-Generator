Project-Password-Generator
Password Generator to randomise a password thats safe and secure. Try out multiple combinations to see what password is right for you.

Requirements
Python must be installed on your computer to run this project.
Please extract Zip file using a tools such as 7-Zip or WinRaR.

Option A: Opening and running via Visual Studio Code

1. Download the repository.
2. Open folder and move to downloads section to find the file Project-Password-Generator-main.
3. Right click and extract all.
4. Click on new folder created and open Password Generator.py.
5. If VS Code opens in "Restricted Mode", click the "Trust" (only if you trust the source of the project).
6. Click the Run button and enjoy the game.

Option B: Run this Using Command Prompt

1. Download this repository.
2. Once downloaded right click the file and extract all.
3. Open the Command Prompt.
4. Navigate to the folder containing Password Generator.py.
5. You can do this by:

   cd Downloads
   cd Project-Password-Generator-main
   cd Project-Password-Generator-main

6. Once Located type py "Password Generator.py". In case this does not work please try: python "Password Generator.py".
